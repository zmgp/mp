# MP 项目

## 功能简介

* 限流
    * 根据IP进行API限流
    * 全局对请求进行限流
    
* 权限管理
    * 用户权限管理
    
* 布隆过滤器
    * 根据 hutool + redis 实现简易布隆过滤器
    
* webSocket
    * 根据 websocket + redis实现
